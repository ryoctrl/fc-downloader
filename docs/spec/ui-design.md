# UI 設計

> UI イメージは別途作成中。確定したらこのドキュメントを正にして実装を合わせる。
> 以下は現スキャフォールドの暫定構成。

## レイアウト

```
┌────────────┬───────────────────────────────────────────────┐
│ Sidebar    │  Content                                       │
│            │                                                │
│ サービス    │  [サービス選択時]                               │
│  Fantia ●  │   ┌──────────────────────┬──────────────────┐  │
│  Fanbox    │   │ WebView(ログイン/閲覧) │ ダウンロード設定  │  │
│  …         │   │                       │  ・同時数         │  │
│            │   │                       │  ・重複スキップ    │  │
│ ライブラリ  │   │                       │  ・種別フィルタ    │  │
│  ビューワー │   │                       │  [開始] 進捗バー   │  │
│            │   └──────────────────────┴──────────────────┘  │
│            │                                                │
│  ⚙ 設定    │  [ビューワー選択時] ツリー一覧                   │
└────────────┴───────────────────────────────────────────────┘
```

## コンポーネント

- `Sidebar` — サービスアイコン一覧（ログイン状態ドット）、ビューワー、設定への遷移。
- `ServicePanel` — 左: `<webview partition="persist:<id>">`、右: ダウンロード設定 + 進捗。
- `ViewerPanel` — `サービス→ユーザー→年→月→投稿` の折りたたみツリー。投稿はフォルダを開ける。
- `SettingsPanel` — 保存先・既定同時数。
