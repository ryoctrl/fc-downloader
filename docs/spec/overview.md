# 概要

## プロダクト

fc-downloader は、ファンクラブ系支援サイトで **自分が支援している** コンテンツを
ローカル PC にダウンロードして保管・閲覧するためのデスクトップアプリ。

対象サイト（予定）:

- Fantia（**MVP 対象**）
- Pixiv Fanbox
- Patreon
- ci-en

## 体験フロー

1. サイドメニューでサービス（Fantia など）を選択。
2. 右側メイン領域に当該サービスのページが内蔵 WebView で表示され、ユーザーがログイン。
   ログイン情報（Cookie）はサービスごとに隔離して永続化される。
3. 同じ画面のダウンロード設定パネルで対象（クリエイター・ファイル種別・同時数・
   重複スキップ）を設定。
4. 「一斉ダウンロード開始」で、支援中コンテンツをまとめて取得。
5. 取得物は内蔵ファイルビューワーで
   `サービス → ユーザー → 投稿年 → 投稿月 → 投稿ID` の階層で一覧・閲覧できる。

## 設計原則

- **ローカル完結**: ネットワーク通信は対象サイトへのアクセスのみ。外部サーバへ
  データを送らない。
- **サービス非依存のコア**: ダウンロードエンジン・ストレージ・UI は特定サイトに依存せず、
  サイトごとの差異は `Service` アダプタに閉じ込める。
- **冪等な再実行**: 何度実行しても、フォルダレイアウトをキーに重複ダウンロードを回避する。
- **個人利用前提**: [security-and-legal.md](security-and-legal.md) を参照。

## スコープ外（現時点）

- クラウド同期・複数端末間共有
- 支援していない / 課金していないコンテンツの取得
- コンテンツの再配布・公開機能
